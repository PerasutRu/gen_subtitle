# การปรับแต่งฟอนต์ Subtitle (Font Customization)

## ภาพรวม
ระบบรองรับการปรับแต่งฟอนต์ subtitle แบบ hard subtitle ให้สวยงามและรองรับหลายภาษา รวมถึงภาษาไทย

## ฟอนต์ที่แนะนำสำหรับภาษาไทย

### ฟอนต์ที่รองรับภาษาไทยได้ดี:
1. **Arial** (แนะนำ) - รองรับภาษาไทยและภาษาอื่นๆ ได้ดี
2. **Helvetica** - สวยงาม รองรับหลายภาษา
3. **Tahoma** - อ่านง่าย รองรับภาษาไทย
4. **Verdana** - ตัวอักษรชัดเจน
5. **DejaVu Sans** - รองรับภาษาหลากหลาย

## ตัวเลือกการปรับแต่ง

### 1. ชื่อฟอนต์ (font_name)
- **ค่าเริ่มต้น**: "Arial"
- **ตัวอย่าง**: "Arial", "Helvetica", "Tahoma"

### 2. ขนาดฟอนต์ (font_size)
- **ค่าเริ่มต้น**: 20 (ปรับลดลงเพื่อความเหมาะสม)
- **แนะนำ**: 18-24
- **เล็ก**: 16-20
- **กลาง**: 20-24
- **ใหญ่**: 24-28

### 3. สีตัวอักษร (font_color)
- **ค่าเริ่มต้น**: "white"
- **ตัวเลือก**: 
  - "white" - ขาว (เหมาะกับวิดีโอมืด)
  - "yellow" - เหลือง (โดดเด่น)
  - "cyan" - ฟ้า
  - "green" - เขียว
  - "magenta" - ม่วงแดง
  - "red" - แดง
  - "blue" - น้ำเงิน

### 4. สีขอบตัวอักษร (outline_color)
- **ค่าเริ่มต้น**: "black"
- **แนะนำ**: "black" (ทำให้อ่านง่ายที่สุด)

### 5. ตัวหนา (bold)
- **ค่าเริ่มต้น**: true
- **ตัวเลือก**: true/false

### 6. ความหนาของขอบ (outline)
- **ค่าเริ่มต้น**: 1.5 (ปรับลดลงให้บางขึ้น)
- **แนะนำ**: 1.0-2.0
- **บาง**: 0.8-1.2
- **กลาง**: 1.5-2.5
- **หนา**: 2.5-3.5

### 7. ความเข้มของเงา (shadow)
- **ค่าเริ่มต้น**: 1.0 (ปรับลดลงให้เบาขึ้น)
- **แนะนำ**: 0.5-2.0
- **เบา**: 0.3-0.8
- **กลาง**: 1.0-1.5
- **เข้ม**: 1.5-2.5

## วิธีใช้งาน

### ตัวอย่าง API Request

```json
{
  "file_id": "your-file-id",
  "language": "th",
  "type": "hard",
  "font_name": "Tahoma",
  "font_size": 20,
  "font_color": "white",
  "outline_color": "black",
  "bold": true,
  "outline": 1.5,
  "shadow": 1.0
}
```

### ตัวอย่างการใช้งานแบบต่างๆ

#### 1. สไตล์มาตรฐาน (เหมาะกับทุกสถานการณ์) - ค่าเริ่มต้นใหม่
```json
{
  "font_name": "Tahoma",
  "font_size": 20,
  "font_color": "white",
  "outline_color": "black",
  "bold": true,
  "outline": 1.5,
  "shadow": 1.0
}
```

#### 2. สไตล์โดดเด่น (เหมาะกับวิดีโอที่มีฉากหลังหลากหลาย)
```json
{
  "font_name": "Tahoma",
  "font_size": 22,
  "font_color": "yellow",
  "outline_color": "black",
  "bold": true,
  "outline": 2.0,
  "shadow": 1.5
}
```

#### 3. สไตล์เรียบง่าย (เหมาะกับวิดีโอที่มีฉากหลังเรียบ)
```json
{
  "font_name": "Arial",
  "font_size": 18,
  "font_color": "white",
  "outline_color": "black",
  "bold": false,
  "outline": 1.0,
  "shadow": 0.5
}
```

#### 4. สไตล์สำหรับภาษาไทย (อ่านง่าย ชัดเจน วรรณยุกต์ไม่โดด)
```json
{
  "font_name": "Tahoma",
  "font_size": 20,
  "font_color": "white",
  "outline_color": "black",
  "bold": true,
  "outline": 1.5,
  "shadow": 1.0
}
```

## คำแนะนำการเลือกฟอนต์

### สำหรับภาษาไทย:
- ใช้ **Tahoma** (แนะนำ) หรือ **Arial** จะได้ผลลัพธ์ที่ดีที่สุด - วรรณยุกต์ไม่โดด
- ขนาดฟอนต์ควรอยู่ที่ **18-22** เพื่อความชัดเจนและไม่ใหญ่เกินไป
- ใช้ **bold: true** เพื่อให้อ่านง่าย
- ขอบควรบางพอดี (**outline: 1.5-2.0**) ไม่หนาเกินไป

### สำหรับภาษาอังกฤษ:
- **Helvetica** หรือ **Arial** ให้ผลลัพธ์สวยงาม
- สามารถใช้ขนาดเล็กกว่าได้ (**16-20**)
- **bold: false** ก็อ่านง่าย

### สำหรับภาษาอื่นๆ:
- **Arial** รองรับภาษาส่วนใหญ่ได้ดี
- ทดสอบกับภาษาที่ต้องการก่อนใช้งานจริง

## การทดสอบ

1. เริ่มต้นด้วยค่าเริ่มต้น (default values)
2. ปรับขนาดฟอนต์ตามความเหมาะสม
3. ทดสอบสีที่เหมาะกับวิดีโอ
4. ปรับความหนาของขอบและเงาตามต้องการ

## ข้อควรระวัง

1. **ฟอนต์ที่ไม่มีในระบบ**: ถ้าระบบไม่มีฟอนต์ที่ระบุ จะใช้ฟอนต์ทดแทนอัตโนมัติ
2. **ขนาดฟอนต์**: ไม่ควรใหญ่หรือเล็กเกินไป (แนะนำ 16-24)
3. **สี**: เลือกสีที่ตัดกับฉากหลังวิดีโอ
4. **ขอบและเงา**: ถ้าหนาเกินไปอาจทำให้อ่านยาก (แนะนำ outline: 1.0-2.0, shadow: 0.5-1.5)

## การแก้ปัญหา

### ฟอนต์ไม่แสดงผลถูกต้อง
- ลองเปลี่ยนเป็น Arial หรือ Helvetica
- ตรวจสอบว่าระบบมีฟอนต์นั้นหรือไม่

### ตัวอักษรอ่านยาก
- เพิ่มความหนาของขอบ (outline)
- เปลี่ยนสีตัวอักษรให้ตัดกับฉากหลัง
- เพิ่มเงา (shadow)

### วรรณยุกต์ภาษาไทยโดด
- ใช้ฟอนต์ **Tahoma** แทน (รองรับภาษาไทยดีที่สุด)
- ลดขนาดฟอนต์ลงเล็กน้อย (18-22)

### ตัวอักษรใหญ่หรือเล็กเกินไป
- ปรับ font_size ให้เหมาะสม (แนะนำ 18-24 สำหรับภาษาไทย)

## ตัวอย่างผลลัพธ์

ระบบจะสร้าง subtitle ที่:
- ✅ รองรับภาษาไทยและภาษาอื่นๆ
- ✅ มีขอบและเงาทำให้อ่านง่าย
- ✅ ปรับแต่งได้ตามต้องการ
- ✅ คุณภาพสูง (CRF 20)
- ✅ จัดตำแหน่งที่ด้านล่างกลาง
- ✅ มี margin ที่เหมาะสม
