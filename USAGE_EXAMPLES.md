# ตัวอย่างการใช้งาน

## การเลือกผู้ให้บริการ

### 1. OpenAI Whisper
**ข้อดี:**
- รองรับหลายภาษา (100+ ภาษา)
- ความแม่นยำสูง
- รองรับไฟล์ขนาดใหญ่
- มี timestamp ที่แม่นยำ

**ข้อเสีย:**
- ค่าใช้จ่ายสูงกว่า
- ต้องใช้ OpenAI API Key

**เหมาะสำหรับ:**
- วิดีโอภาษาต่างประเทศ
- วิดีโอที่ต้องการความแม่นยำสูง
- วิดีโอที่มีหลายภาษาปะปน

### 2. Botnoi Gensub
**ข้อดี:**
- เหมาะสำหรับภาษาไทย
- ราคาถูกกว่า (ขึ้นอยู่กับแพ็คเกจ)
- รองรับภาษาในอาเซียน

**ข้อเสีย:**
- รองรับภาษาน้อยกว่า
- อาจมีข้อจำกัดด้านขนาดไฟล์

**เหมาะสำหรับ:**
- วิดีโอภาษาไทย
- วิดีโอภาษาในอาเซียน
- งบประมาณจำกัด

## ตัวอย่างการใช้งาน

### ตัวอย่างที่ 1: วิดีโอภาษาไทย
```
1. อัปโหลดวิดีโอ
2. เลือก "Botnoi Gensub" สำหรับแกะเสียง
3. แก้ไข subtitle ตามต้องการ
4. แปลเป็นภาษาอังกฤษด้วย "Botnoi" หรือ "OpenAI"
```

### ตัวอย่างที่ 2: วิดีโอภาษาอังกฤษ
```
1. อัปโหลดวิดีโอ
2. เลือก "OpenAI Whisper" สำหรับแกะเสียง
3. แก้ไข subtitle ตามต้องการ
4. แปลเป็นภาษาไทยด้วย "OpenAI"
```

### ตัวอย่างที่ 3: วิดีโอหลายภาษา
```
1. อัปโหลดวิดีโอ
2. เลือก "OpenAI Whisper" สำหรับแกะเสียง
3. แก้ไข subtitle ตามต้องการ
4. แปลเป็นหลายภาษา:
   - อังกฤษ: OpenAI
   - ลาว: Botnoi
   - พม่า: Botnoi
   - เวียดนาม: Botnoi
```

## Tips & Tricks

### การเลือก Provider สำหรับการแปล

**ใช้ OpenAI เมื่อ:**
- ต้องการ style การแปลที่ซับซ้อน
- ต้องการความแม่นยำสูง
- แปลเป็นภาษาที่ไม่ใช่ภาษาในอาเซียน

**ใช้ Botnoi เมื่อ:**
- แปลภาษาในอาเซียน (ไทย, ลาว, พม่า, กัมพูชา, เวียดนาม)
- ต้องการประหยัดค่าใช้จ่าย
- แปลเนื้อหาทั่วไป

### การกำหนด Style Prompt

**ตัวอย่าง Style Prompt:**
- "แปลให้เป็นทางการ เหมาะสำหรับการนำเสนอ"
- "ใช้คำง่ายๆ เข้าใจง่าย"
- "แปลแบบสบายๆ เป็นกันเอง"
- "ใช้คำศัพท์ทางเทคนิค"
- "แปลให้เหมาะสำหรับเด็ก"

### การแก้ไข Subtitle

1. **ใช้ Video Player**: คลิกที่ timestamp เพื่อเล่นวิดีโอจากจุดนั้น
2. **ตรวจสอบความถูกต้อง**: ฟังเสียงและแก้ไขข้อความที่ผิด
3. **ปรับ Timing**: แก้ไข start/end time ถ้าจำเป็น
4. **บันทึกก่อนแปล**: บันทึกการแก้ไขก่อนไปขั้นตอนแปลภาษา

## การแก้ปัญหา

### ปัญหา: แกะเสียงไม่ถูกต้อง
**วิธีแก้:**
1. ลองเปลี่ยน provider (OpenAI ↔ Botnoi)
2. ตรวจสอบคุณภาพเสียงในวิดีโอ
3. แก้ไข subtitle ด้วยตนเองในขั้นตอนแก้ไข

### ปัญหา: การแปลไม่ตรงความหมาย
**วิธีแก้:**
1. เพิ่ม style prompt ที่ชัดเจน
2. ลองเปลี่ยน provider
3. แปลใหม่ด้วย style prompt ที่แตกต่าง

### ปัญหา: Subtitle ไม่ sync กับวิดีโอ
**วิธีแก้:**
1. แก้ไข timing ในขั้นตอนแก้ไข subtitle
2. ตรวจสอบว่าวิดีโอต้นฉบับไม่มีปัญหา
3. ลองแกะเสียงใหม่

## Best Practices

1. **เลือก Provider ที่เหมาะสม**: ใช้ Botnoi สำหรับภาษาไทย, OpenAI สำหรับภาษาอื่น
2. **แก้ไข Subtitle ก่อนแปล**: ตรวจสอบและแก้ไข subtitle ต้นฉบับให้ถูกต้องก่อนแปล
3. **ใช้ Style Prompt**: กำหนด style ที่ชัดเจนเพื่อผลลัพธ์ที่ดีกว่า
4. **ทดสอบหลาย Provider**: ลองใช้ทั้ง OpenAI และ Botnoi เพื่อเปรียบเทียบผลลัพธ์
5. **บันทึกไฟล์ SRT**: ดาวน์โหลดและเก็บไฟล์ SRT ไว้เป็นสำรอง
